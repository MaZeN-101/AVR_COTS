# AVR_COTS
Implementing most of the peripheral drivers of AVR Microcontroller (MCAL Layer):

1. DIO
2. PORT
3. GIE
4. EXTI
5. ADC
6. UART
7. SPI
8. TWI

Interfacing with HW devices (HAL Layer):

1. LED 
2. Seven Segments
3. SWITCH
4. Character LCD
5. Keypad
6. DC Motor
7. Stepper Motor
8. EEPROM

(Library Layer) :

1. Standard Types
2. Bit Math
